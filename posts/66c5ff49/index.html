如何使用HomeBrew？

基本用法

HomeBrew安裝完成之後就可以使用它進行軟體套件的安裝管理ㄌ！以下是幾個常用的命令：

  • 安裝軟體套件：brew install [packge-name] ，例如安裝chrome：brew install chrome
  • 更新已安裝的軟體套件：brew upgrade [package-name]，例如更新spotify：brew upgrade spotify
  • 列出已安裝的軟體套件：brew list
  • 查看軟體套件的訊息：brew info [package-name]，例如：brew info ffmpeg
  • 搜尋可用的軟體套件：brew search [keyword]，例如：brew search python

所以如果我要使用HomeBrew安裝Telegram Desktop我只需要：

  1. 打開終端機
  2. 輸入brew install telegram-desktop

就成功安裝了！

__＊注意如果軟體名稱裡有空格，請將空格替換成 - __
